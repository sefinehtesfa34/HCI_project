                    About NADLE

                    The National Academic Digital Library of Ethiopia (NADLE) is a project under Ministry of Science and Higher Education, Ethiopia. The objective is to collect and collate metadata and provide full text index from several national and international digital libraries, as well as other relevant sources. It is a academic digital repository containing textbooks, articles, audio books, lectures, simulations, fiction and all other kinds of learning media. The NADLE provides free of cost access to many academic books in English and the Other languages.

                    Bulk Registration

                    To register the users of your institute through "Bulk Registration" in NDLI portal, kindly send the excel file in comma separated values (CSV) format to ndl-support@iitkgp.ac.in. Please mention the institute name within the mail.

                    The steps for creating CSV file are as follows:

                    1. Create an excel sheet of your users which includes students, faculties and staffs.
                    2. Excel file should contain (1st column) e-mail id (2nd column), first name (3rd column), last name only with no other information like serial number, heading, signature, etc.
                    3. Save the excel file in .csv format.

                    Note: One csv file can contain upto 1000 records. If required, you can create multiple files.
